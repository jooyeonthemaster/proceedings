외주개발 결론

마케팅은 빼고, 인맥·영업 동선·회사 소개서에 포트폴리오 한 페이지 — 분기 1건을 보수적으로 잡는다.

5가지 결론
  • 인맥 기반 영업 유지 — 기존 방식 그대로 간다.
  • 네트워크 크로스셀 — 재영 기존 네트워크 + 주희 포토부스 영업 동선에서 자연스럽게 니즈 캐치.
  • 소개서 포트폴리오 페이지 — 회사 소개서에 외주개발 포트폴리오 페이지 하나 추가하는 정도.
  • 마케팅팀 리소스 투입 없음 — 별도 리소스 배정하지 않는다.
  • 재무 계획은 분기 1건(500만원)으로 보수적 계상.
