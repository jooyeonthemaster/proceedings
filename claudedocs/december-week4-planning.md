### 핵심 메시지

- 지난 3주차 회의에서 논의한 **VS AI 유튜브 콘텐츠** 방향 → **전원 동의**
- 단순 개발 외주, 전시 대행을 넘어 **창작 집단**으로 포지셔닝
- 콜드메일, 웜메일 영업 → **유튜브 콘텐츠로 자체 네임밸류 확보** → **기업이 먼저 찾아오는 구조**

### 핵심 전환점

| Before          | After                              |
| --------------- | ---------------------------------- |
| B2B 영업이 메인 | 유튜브 브랜딩이 메인, B2B는 부차적 |
| 개발 외주 수주  | 창작 집단으로 작품 판매            |
| 콜드메일/웜메일 | 콘텐츠 바이럴 → 인바운드 영업     |

### VS AI 콘텐츠의 3가지 가치

1. **영업용 콘텐츠** - 기업에게 우리 역량 자연스럽게 노출
2. **브랜딩 콘텐츠** - 네안데르를 알리는 홍보 채널
3. **자체 IP** - 재밌는 콘텐츠로서 독립적 가치 보유

### 제목

**우리의 롤모델: 팀랩 & 디스트릭트**

### 유사 기업 분석

| 기업                 | 시작                                                         | 현재                                                  |
| -------------------- | ------------------------------------------------------------ | ----------------------------------------------------- |
| **팀랩**       | B2B 미디어 콘텐츠 개발 외주, 스마트폰 앱 개발, 웹사이트 개발 | 오프라인 콘텐츠 기반 체험 프로그램**창작 집단** |
| **디스트릭트** | 미디어 콘텐츠 개발                                           | "파도" 등**작품**으로 세계적 인지도             |

- 팀랩, 디스트릭트 모두 **초기에는 B2B 외주**로 시작
- **오프라인 체험 콘텐츠**가 메인이 되면서 **창작 집단**으로 진화
- 우리도 같은 경로: **B2B 영업은 부차적, 콘텐츠 창작이 메인**

## 슬라이드 8: 네안데르랩 - 우리만의 차별점

### 제목

**긱블도, 스펀지도 아니다. 창작 집단이다.**

### 포지셔닝 비교

| 유형                | 예시                       | 우리의 방향 |
| ------------------- | -------------------------- | ----------- |
| 교육/실험 콘텐츠    | 긱블, 스펀지               | ❌ 아님     |
| **창작 집단** | **팀랩, 디스트릭트** | ✅ 이것     |

### AI 콘텐츠의 작품화

- 팀랩의 "작품", 디스트릭트의 "파도"처럼
- 우리 AI 콘텐츠도 **하나의 작품으로 브랜딩** 필요
- 엄청난 예술적 가치가 아니라, **콘텐츠 자체로 가치를 보유**
- 기업 입장: "내 행사에 이런 프로그램이 있으면 좋겠다"

### 장기 비전

```
10개 이상의 오프라인 체험 프로그램 개발
         ↓
하나의 공간에 모아서 전시 콘텐츠로 런칭
         ↓
전 세계를 돌아다니며 AI 콘텐츠 전시
         ↓
"AI 콘텐츠 개발사"가 아닌 "AI 콘텐츠 창작 집단"
```

## 슬라이드 9: 네안데르랩 - 유튜브 전략

### 제목

**유튜브: 브랜딩의 창구**

### 유튜브의 역할

> 유튜브로 돈을 버는 게 목적이 아니다

| 유튜브의 역할       | 설명                                     |
| ------------------- | ---------------------------------------- |
| **작품 홍보** | 우리의 AI 콘텐츠 작품들을 세상에 알림    |
| **브랜딩**    | 네안데르라는 이름을 각인시킴             |
| **바이럴**    | 콘텐츠가 퍼지면서 자연스러운 인지도 확보 |

### 기대 효과

```
유튜브 콘텐츠 바이럴
      ↓
네안데르 네임밸류 확보
      ↓
기업들이 자연스럽게 우리에게 연락
      ↓
인바운드 영업 구조 완성
```

### 콘텐츠 톤앤매너

- ❌ 진지하고 딱딱한 톤
- ✅ **진짜 또라이들처럼** 재밌고 유쾌하게 접근
- ✅ 미친 사람들 마냥 길거리로 나가서 체험시키고 튀는 형태

## 슬라이드 10: VS AI - 스트릿 컨셉

### 제목

**오프라인을 넘어, 스트릿(길거리)로**

### 핵심 컨셉

> AI 콘텐츠를 오프라인, 그것을 넘어서 **스트릿(길거리)**로 가져온다

### 스트릿 컨셉 예시

| 콘텐츠      | 스트릿 버전                                                       |
| ----------- | ----------------------------------------------------------------- |
| AI 포토부스 | **스트릿 포토그래퍼** - AI 포토 로봇을 길거리로 가져와 체험 |
| AI 사주     | **푸드트럭 사주집** - 푸드트럭을 AI 사주집으로 꾸며서 운영  |
| AI 판사     | 길거리 재판 체험?                                                 |
| AI 기자     | 기자회견장에서 AI 마네킹으로 인터뷰?                              |

### 푸드트럭 사주집 아이디어

- 푸드트럭을 AI 사주집 형태로 꾸밈
- 길에서 고객들에게 AI 사주 체험 제공
- **바로 옆에 실제 사주가 배치**
- 고객들에게 **만족도 비교 체험** 제공 → VS AI 컨셉 극대화

### 영감: 뱅크시

- 뱅크시의 스트릿 아트 컨셉 차용
- 전시장이 없어도 **길거리가 우리의 무대**
- 미친 척 사람들에게 보여주고 튀는 형태

## 슬라이드 11: VS AI - 신비로움 전략

### 제목

**마스크로 가린 정체, 신비로운 창작 집단**

### 문제 인식

- 팀원 전원 **나이가 어림**
- **경력 자체가 길지 않음**
- → 신빙성이 떨어질 수밖에 없음

### 해결 전략

```
┌─────────────────────────────────────┐
│     마스크로 신상 정보 철저히 가림   │
│              ↓                      │
│      신비로운 느낌 극대화           │
│              ↓                      │
│    유쾌하고 친근하지만 정체불명     │
│              ↓                      │
│       궁금증 유발 → 팬층 형성       │
└─────────────────────────────────────┘
```

### 톤앤매너 균형

- **유쾌하고 친근한** 콘텐츠 톤
- BUT **정체는 신비롭게** 유지
- 아무리 재밌어도 **나이/신상 노출 금지**

## 슬라이드 29: 악센트 WOW - 가격 이슈

### 제목

**악센트 WOW: 가격으로 인한 이탈 문제**

### 현황 분석

| 일반 생일 카페 | 악센트 WOW                           |
| -------------- | ------------------------------------ |
| 커피 6,000원   | 페이퍼 사쉐**1.5만 원** (최저) |
| -              | 50ml 향수**3.8만 원** (최고)   |

### 문제

- 고객층: 대부분 **10대 소녀** (돈이 없음)
- 약 **70%**가 가격을 듣고 **구매 포기 후 이탈**
- 그냥 생일 이벤트 중 하나로 방문했을 뿐, **향수에 대한 가치 인식 없음**

### 핵심 인사이트

> 악센트 WOW에서는 **"향수"라는 상품에 대한 가치는 전혀 없다.**
> 사람들은 향수를 구매하기 위해 방문하는 게 아니라,
> 홍대에서 진행되는 수많은 생일 이벤트 중 하나로 방문하는 것.

## 슬라이드 30: 악센트 WOW - 데이터 수집 제안

### 제목

**제안: 이탈 고객 데이터 수집**

### 제안 내용

> 알바생(이우빈)에게 요청:
> **비싸서 가격을 듣고 돌아가는 고객 수를 기록**

### 분석 목적

**가설 검증**

```
현재: 10명 중 3명만 구매 (1.5만 원) = 4.5만 원
만약: 5,000원 상품 출시 시
      10명 중 10명 구매 = 5만 원

→ 매출 증가 가능성 검증
```

**참고**

- 3.8만 원 구매자는 어차피 **비싼 걸 살 사람**
- 문제는 1.5만 원도 부담스러워서 돌아가는 **70%**

### 주의

> 바로 저렴한 버전을 출시하자는 게 아님
> 일단 **최소 한 달 데이터**로 분석 필요

## 슬라이드 32: 스팟 - 브랜드 소개

### 제목

**스팟(SPOT): 공간의 향기를 담은 디퓨저 브랜드**

### 브랜드명 의미

> **S**tory + **P**ot + Scen**T** = **SPOT**

### 콘셉트

- 공간의 향기를 설계
- 화분 모양의 **팟(Pot)**에 담아 디퓨저 제작
- 다양한 **공간들과 콜라보**
- 해당 공간의 **스토리를 표현하는 향기**를 화분에 담아냄

### 비즈니스 모델

```
다양한 핫플레이스들과 콜라보
           ↓
해당 공간의 스토리를 표현하는 향기 설계
           ↓
화분 디퓨저로 제작
           ↓
온라인 + 오프라인 판매
```

### 포지션

> 네안데르랩과 함께 **장기 비전** 사업
> (캐시카우인 악센트 ID/WOW와 구분)
