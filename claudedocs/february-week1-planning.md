- **날짜**: 2026년 2월 1주차
- **주제**: 2026년 1분기 KPI 설정 & 악센트 아이디 제품 전략

## 전체 슬라이드 구성 (총 18개)

### Part 1: 개발팀 1분기 KPI (슬라이드 1-9)

#### 슬라이드 1: 타이틀
- **제목**: 2026년 1분기 KPI 설정
- **부제**: 개발팀 & 사업기획팀
- **핵심 통계**:
  - 개발팀 2명 (김주연, 김제연)
  - 사업계획서 2억 목표
  - 뿌덕 프로그램 분기별 1개
  - 4개 법인 지원사업 도전

#### 슬라이드 2: 개발팀 구성
- **개발팀 구성원**
  - 김주연 (CTO): 내부 개발 총괄, 사업계획서 병행
  - 김제연 (프리랜서): 외주 개발 담당, 내부 상품 개발 협력
- **핵심 목표**: 영업 들어오는 개발건 전량 처리

#### 슬라이드 3: 외주 개발 KPI
- **현실**:
  - 영업팀 → 개발건 → 김제연 처리
  - 시키는 대로 해야 하는 구조
- **KPI 설정 방향**:
  1. 납기 준수율 100%
  2. 고객 만족도 조사 (재계약율)
  3. 개발 효율화 (재사용 가능 모듈 축적)
  4. 기술 문서화율

#### 슬라이드 4: 내부 개발 KPI - 회사 협업
- **영업팀 협업**:
  - 제안 가능한 프로그램 종류 확대
  - 신규 솔루션 1개 이상 개발
- **마케팅팀 협업**:
  - 콘텐츠 일별 기록 자동화
  - 댓글 수집 시스템
  - 언급 게시물 크롤링
  - LLM 인사이트 도출 대시보드
- **재무회계팀 협업**:
  - 악센트 아이디 뿌덕 v2 연동
  - 향료 데이터 관리
  - 재고 데이터 효율화

#### 슬라이드 5: 내부 개발 KPI - 뿌덕 서비스
- **분기별 프로그램 1개 개발**
  - 단순 AI 추천 넘어서기
  - "확실하게 다른 프로그램" 체감
  - 더욱 즐길 수 있는 형태
- **1분기 목표**: 새로운 프로그램 1개 출시

#### 슬라이드 6: 1분기 추가 KPI
- **포토부스 로봇 설치** (김제연 협업)
  - 악센트 아이디 매장 설치 완료
- **향수 키링 케이스 파츠**
  - 10ml 기준 30개 디자인 제작
- **3D 커스터마이징 웹 서비스**
  - 케이스티파이 스타일 참고
  - 온라인에서 향수 키링 커스터마이징

#### 슬라이드 7: 연간 목표
- **2026년 연간 KPI**:
  1. 뿌덕 프로그램 총 4개 개발 (분기별 1개)
  2. 글로벌 비즈니스 확장
  3. 고객 1만명 데이터 수집
  4. 자체 분석 알고리즘 강화
     - 현재: 이미지 기반 추천
     - 목표: 실제 만족도 기반 알고리즘

### Part 3: 악센트 아이디 제품 전략 (슬라이드 11-16)

#### 슬라이드 11: 제품 전략 타이틀
- **핵심 인사이트**: 제품 = 케이스
- 시그니처 케이스가 가치를 창출

#### 슬라이드 12: 시그니처 케이스의 힘
- **Before**: 그냥 10ml 향수 = 가치 없음
- **After**: 시그니처 케이스 = 가치 창출
- **결론**: 케이스가 브랜드 아이덴티티

#### 슬라이드 13: 콜라보 & 트렌드 전략
- **콜라보 = 핵심 전략** (케이스티파이 벤치마킹)
- **트렌드 기반 상품 기획**:
  - 두쫀쿠 향수
  - 스크류바 향수
  - @uglyperfume 스타일
- **디퓨저 라인업**:
  - 화분 모양 계속 변경
  - 시즌별 런칭

#### 슬라이드 14: AI 프로그램 전략
- **AI = 유입 수단 + 아이덴티티**
  - 재밌는 프로그램으로 사이트 방문 유도
  - 다양한 향 조합별 만족도 기록
  - 지속적 업그레이드
- **마이페이지 피드백 기능**
  - 구매 후 피드백 기록
  - AI 개선 향 제공

#### 슬라이드 15: 제품 라인업 확장
- **기존**: 10ml, 50ml, 뿌덕 디퓨저
- **신규 추가**:
  - 시향지 세트 (5,000원)
  - AI 추천 향 체험용
- **AI 커스터마이징**:
  - 파츠 자동 조합
  - scentalytics.com 참고
