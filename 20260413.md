## 전사 3대 사업 우선순위

### 제1목표: 악센트 아이디 오프라인 + 와우 매출 유지 & 온라인 활성화

**핵심:** 월 매출 2,500만원(오프라인 1,500만 + 와우 1,000만) 사수 + 온라인 매출 월 500만원 목표 달성

이것이 5~7월 생존의 기반이다. 악센트 매출이 흔들리면 런웨이가 더 빨리 소진되고, 7월 말 판단 자체가 불가능해진다. 마케팅팀(선화+정연)이 기존 업무에 집중하면서 온라인 매출을 끌어올리는 것이 핵심.

**담당:** 유선화(CMO) + 김정연 — 악센트 아이디 오프라인/온라인 마케팅 유지·성장
**운영:** 이동주 — 매장 운영 관리, 재무 관리 유지

### 제2목표: AI 포토부스 B2B — 축제·행사 활성화 전략

**핵심:** 명확한 목표 + 정확한 영업 전략 수립

포토부스는 상품 개발이 이미 완료된 상태다. 제안서(21p)도 주희가 완성했다. 지금 필요한 것은 "영업 전략의 구체화"다.

**영업 타겟 우선순위:**

| 순위 | 타겟 | 시급성 | 단가 | 핵심 접근법 |
|------|------|--------|------|-------------|
| 1순위 | 지자체 축제 | 5~6월 봄 축제 막차 | 중~고 | 나라장터 모니터링 + 구청/시청 담당자 콜드콜 |
| 2순위 | 대학 축제 | 5월 대축제 시즌 (다소 늦음) | 저~중 | 총학생회 인스타 DM + 전화 동시 접근 |
| 3순위 | 기업 행사/박람회 | 중기 | 고 (300~500만+) | 링크드인 / 대표번호 경유 |

**핵심 원칙:**
- 기존 완성된 상품 기준으로 영업 먼저
- 추가 기능 개발은 학원 AI MVP 안정화 후
- 첫 1~2건은 원가 수준이라도 레퍼런스 확보가 우선
- 주희가 포토부스 영업 동선에서 외주개발 니즈도 자연스럽게 캐치 (크로스셀)

**담당:** 김주희(영업 실행) + 유재영(큰 건 동행) + 이동주(현장 운영/견적)

## 학원 AI 솔루션 — 실행 계획 상세

### 연간 목표

**2026년 월 매출 목표: 3,700만원**

이 목표를 달성하기 위한 과금 모델(독점 구독 vs 크레딧)은 아직 확정하지 않는다. 철저하게 고객 반응 및 수요 조사를 기반으로 판단한다.

**과금 모델 후보:**

| 모델 | 구조 | 월 3,700만 달성 조건 | 장점 | 단점 |
|------|------|---------------------|------|------|
| A. 독점 구독 | 구별 독점 계약, 월 50만원 | 74개 학원 | 안정적 MRR, FOMO 영업 가능, 이탈률 낮음 | 초기 영업 난도 높음, 가격 저항 |
| B. 크레딧 기반 | 사용량 기반 과금 | 학원 수 × 평균 사용량 | 진입장벽 낮음, 빠른 확산 | MRR 변동성, 이탈 쉬움 |
| C. 하이브리드 | 기본료 + 크레딧 | 기본료 확보 + 사용량 업셀 | 양쪽 장점 결합 | 과금 구조 복잡 |

**핵심 원칙:** 4월 시장 조사 → 데이터 기반 1차 판단 → 5월 출시 모델 확정

### 시장 검증 전략 — 시나리오 기반 접근

모든 시나리오를 사전에 정의하고, 가지치기(decision tree) 방식으로 설계해서 접근한다. 감이 아니라 데이터로 판단한다.

#### 시나리오 트리 설계

```
[4월 시장 조사]
├── 영업 채널: 유재영 직접 방문 (대흥 학원 10곳)
│   ├── 테스트 A: 가격 먼저 제시 후 기능 소개
│   │   ├── 반응: 긍정 → 가격 수용도 높음, 독점 모델 유력
│   │   ├── 반응: 기능에만 관심 → 크레딧 모델 검토
│   │   └── 반응: 거부 → 가격대 재조정 필요
│   ├── 테스트 B: 기능 먼저 소개 후 가격 제시
│   │   ├── 반응: "얼마예요?" 먼저 물어봄 → 구매 의향 높음
│   │   ├── 반응: 기능에 감탄하지만 가격에 주저 → 무료 체험 후 전환 전략
│   │   └── 반응: 기능 자체에 무관심 → 제품 피벗 필요
│   └── 테스트 C: 가격 언급 없이 무료 체험만 제안
│       ├── 반응: 적극 수용 → 체험 후 전환율 측정
│       ├── 반응: "무료면 해볼게요" 수준 → 진짜 니즈 vs 호의 구분 필요
│       └── 반응: 거부 → 시간 투자조차 안 하는 것 = 니즈 없음
│
└── 마케팅 채널: 유선화 인스타 광고 (2개 콘텐츠)
    ├── 콘텐츠 1안 (시장 조사형): 무료 체험 + 학원 정보 수집
    │   ├── 반응: 신청 많음 → 시장 니즈 확인, 미팅 파이프라인 확보
    │   ├── 반응: 신청 적음 → 콘텐츠/타겟팅 문제 vs 니즈 부재 구분
    │   └── 측정 지표: 신청 수, 학원 정보 제출률, 미팅 전환율
    └── 콘텐츠 2안 (사전예약형): 독점 이용권 50만원 사전 예약
        ├── 반응: 예약 발생 → 50만원 가격 검증 완료, 독점 모델 확정
        ├── 반응: 관심은 있으나 예약 안 함 → 가격 저항, 체험 먼저 필요
        ├── 반응: 무반응 → 콘텐츠/타겟 문제 or 가격 과다
        └── 측정 지표: 예약 수, CPA, 문의 수, 랜딩 이탈률
```

#### 4월 말 1차 판단 기준

| 판단 항목 | 독점 모델 확정 조건 | 크레딧 모델 전환 조건 | 피벗 필요 조건 |
|-----------|---------------------|----------------------|----------------|
| 직접 미팅(10곳) 구매 의향 | 3곳 이상 "50만원 OK" | 5곳 이상 "기능은 좋은데 비쌈" | 7곳 이상 무관심 |
| 인스타 2안(50만원) 사전 예약 | 3건 이상 예약 | 문의만 있고 예약 0 | 노출 대비 클릭률 1% 미만 |
| 인스타 1안(무료체험) 신청 | 10건 이상 + 미팅 전환 3건+ | 신청 있으나 미팅 전환 낮음 | 5건 미만 |
| 기능별 반응 | 3개 이상 기능에 "이거 꼭 필요" | 1~2개만 반응 | 전반적 무관심 |

### 유선화 (CMO) — 마케팅 & 시장 반응 테스트

**역할:** 학원 AI 솔루션 인스타그램 마케팅 콘텐츠 기획·제작·광고 집행. 고객 반응 데이터 수집 및 분석.

**4월 핵심 미션:** 4월 20일까지 인스타 계정 개설 + 2개 콘텐츠 제작 및 광고 배포

**콘텐츠 상세 기획:**

#### 콘텐츠 1안 — 시장 조사형 (무료 체험 + 인사이트 확보)

**목적:** 시장 니즈 확인 + 잠재 고객 정보 확보 + 미팅 파이프라인 구축

**컨셉:** "AI가 시험지를 만들어준다? 선생님, 직접 체험해보세요"

**콘텐츠 구성:**
- 시연 영상 기반 (주연이 출발 전 제공하는 화면 녹화 소스 활용)
- AI 문제 출제 과정 하이라이트 (지문 업로드 → AI 분석 → 시험지 생성)
- "무료 체험 신청하기" CTA

**수집 정보 (신청 폼):**
- 학원명
- 학원 위치 (구/동)
- 학원 규모 (학생 수 대략)
- 주요 과목
- 원장/선생님 연락처
- 희망 미팅 일정
- 현재 시험지 제작 방식 (수기/기존 솔루션/없음)

**광고 타겟팅:**
- 지역: 서울 전역 (구별 성과 비교)
- 관심사: 학원, 교육, 입시, 내신
- 연령: 30~55세
- 예산: 일 2~3만원 × 10일 = 20~30만원

**성공 지표:**
- 신청 10건 이상
- 미팅 전환 3건 이상
- 구별 반응 데이터 확보

#### 콘텐츠 2안 — 사전 예약형 (50만원 가격 검증)

**목적:** 50만원이라는 가격에 대한 실제 시장 반응 확인. 사전 고객 확보.

**컨셉:** "우리 구에 딱 3자리. AI 학원 솔루션 독점 이용권 사전 예약 모집 중"

**콘텐츠 구성:**
- 상세 기능 소개 영상 (학원 ERP + AI 문제 출제 + 내신 분석)
- 월 50만원 독점 이용권 가격 명시
- "구별 3곳 한정, 사전 예약 모집 중" FOMO 프레이밍
- "사전 예약하기" CTA → 예약 폼 또는 DM 유도

**상세 기능 쇼케이스 (영상 내 포함):**
1. 학원 ERP 시스템 — 학생 관리, 출결, 상담 기록
2. AI 내신 지문 분석 — 교과서 지문 업로드 → AI 핵심 포인트 추출
3. AI 시험 문제 생성 — 난이도별 자동 출제, 객관식/서술형
4. AI 시험지 자동 조판 — 문제 선택 → 시험지 PDF 즉시 생성
5. 학습 데이터 리포트 — 학생별 취약 단원 분석

**광고 타겟팅:** 1안과 동일 세팅으로 AB 비교

**성공 지표:**
- 사전 예약 3건 이상 → 50만원 가격 검증 완료
- 문의 10건 이상 → 관심은 있으나 가격 조정 검토
- 무반응 → 콘텐츠 리웍 or 가격 재설정

### 유재영 (CEO) — 영업 & 현장 인사이트 수집

**역할:** 학원 AI 솔루션 직접 영업. 대흥 지역 학원 10곳 미팅을 통한 시장 검증. 가설 기반 체계적 테스트.

#### 영업 접근 전략 — 가설 기반 테스트 설계

단순히 "지불 의사 있어요?" 수준이 아니라, 사전에 가설을 설정하고 여러 경우의 수를 나눠서 접근한다. 디테일한 반응을 전부 기록한다.

**테스트 변수 매트릭스:**

| 변수 | 케이스 A | 케이스 B | 케이스 C |
|------|----------|----------|----------|
| **가격 제시 타이밍** | 초반에 "월 50만원" 먼저 제시 | 기능 전부 보여준 후 마지막에 제시 | 가격 언급 안 함, 관심도만 확인 |

### 이동주 (재무/운영) — QA & 인사이트 기반 개선

**역할:** UI/UX QA 총괄. 유선화·유재영이 수집한 인사이트를 제품 개선 방향으로 정리. 문제 퀄리티 검수.

**4월 핵심 미션:** 학원 AI 솔루션의 UI/UX QA + 문제 퀄리티 검수 + 인사이트 기반 개선안 정리

#### QA 영역 상세

**1. UI/UX QA**

학원 선생님이 실제로 쓸 때 막히는 부분이 없는지 철저하게 검수한다.

검수 항목:
- 첫 화면 진입 시 바로 뭘 해야 하는지 직관적인가
- 지문 업로드 → 문제 생성까지 몇 클릭인가 (최소화)
- 에러 발생 시 사용자가 이해할 수 있는 메시지인가
- 모바일/태블릿에서도 정상 작동하는가
- 로딩 시간이 체감상 길지 않은가
- 결과물(시험지) 다운로드/인쇄가 원활한가
- 학원 선생님(IT 비전문가) 관점에서 용어가 이해 가능한가

**2. 문제 퀄리티 검수**

AI가 생성한 문제가 실제 내신 시험에 쓸 수 있는 수준인지 검수한다.

검수 항목:
- 문제의 정확성 (오답이 없는지)
- 난이도 구분이 적절한지
- 지문과 문제의 연관성
- 선택지 구성의 적절성 (너무 쉬운 오답, 중복 선택지 등)
- 서술형 문제의 채점 기준 명확성
- 교과 과정 범위 내인지
- 실제 내신 출제 패턴과 유사한지

**3. 인사이트 기반 QA 리스트 관리**

유선화(광고 반응)와 유재영(미팅 인사이트)이 수집한 고객 피드백을 QA 리스트로 전환한다.

```
유재영 미팅 인사이트 → 이동주가 QA 항목으로 변환
예시:
  "원장이 시험지 양식이 우리 학원 양식이랑 달라서 못 쓴다고 함"
  → QA 항목: 시험지 양식 커스터마이징 가능 여부 확인
  → 개선안: 학원별 양식 템플릿 기능 추가 검토

유선화 광고 반응 → 이동주가 개선 방향으로 정리
예시:
  "광고 댓글에 '수학도 되나요?' 질문 3건"
  → 개선안: 수학 과목 지원 우선순위 상향 검토
```
